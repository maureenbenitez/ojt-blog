March 3, 2025

March

Coding

We are starting our fifth week of OJT today, and with March now upon us, it marks the second month of our training. When we arrived, we learned that there was no internet connection. ICTO also had no internet, so we didn’t bother going there. We were told that it would be fixed soon, so we waited. By afternoon, the internet came back, but it was slow and kept disconnecting. Unfortunately, we weren’t able to make much progress on our coding tasks. However, we did go over the tasks that we’ll be working on, as well as the ones that aren’t finished yet, which we will continue on the next coding day once the internet is stable.

March 4, 2025

March

Coding

The internet connection is better now, so we continued coding today. Kent finished fixing the template to be used for exportation. We then started working on the backend for exporting. However, my task for the day was to refine the "View Students" function, which I successfully completed. As for the backend of exporting, we haven’t figured it out yet and are still encountering errors.

March

Coding

March

Clerical Work

On the same day, we downloaded the audio files and voice recordings to prepare for the assigned URO flag ceremony on Monday.
